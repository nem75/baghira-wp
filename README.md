# Baghira Homepage

Website based on Wordpress for the Tribal Style Dance group Baghira.

## Development Environment

Copy an existing `.env` file to project root or create it akin to `env.sample`.

- Prepare: `docker-compose -f docker-compose.base.yaml build && docker-compose build`
- Start: `docker-compose up`

Site runs on http://localhost

Database is reachable on local port 3306.
